# Vuetify Todo
*Vue 2*

Stand-alone web app to track tasks

## Features
- Add, mark as done and delete tasks
- Tabs available via side bar
- Fancy clock

## Technologies
- HTML, CSS, JS
- Vue 2.6
- Vue-router 3.5
- Vuetify 2.6.0

## [Live Demo](https://apalevich.github.io/vuetify-todo/)
